# PowerFlex Gym - Fitness Web Application

### Project Summary
PowerFlex Gym is a fully responsive fitness website designed to showcase world-class gym facilities, expert trainers, and specialized workout programs. The application serves as a digital hub for potential members to explore membership options, view class schedules, and connect with the fitness community.

### Major Features
* **Responsive Program & Trainer Showcase:** A dynamic grid layout using Flexbox/Grid to display fitness programs (Strength, Cardio, Yoga) and trainer profiles with hover effects.
* **Interactive JavaScript Components:** Includes an automated image slideshow, a testimonials carousel, and animated numerical counters for gym achievements.
* **Mobile-First Navigation:** Features a fully functional hamburger menu and navigation bar that adapts seamlessly to different screen sizes.

---

### Core Technologies
* **HTML5** (Semantic structure)
* **CSS3** (Custom properties, Flexbox, Grid, Media Queries)
* **JavaScript ES6+** (DOM manipulation for sliders and counters)
* **FontAwesome** (Iconography)
